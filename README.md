# Espejo

Diario personal privado. Cifrado E2E, hábitos, analítica.

## Setup

```bash
pnpm install
pnpm dev
```

## Sync (opcional)

Si quieres sincronizar entre dispositivos:

1. Crea proyecto en [Supabase](https://supabase.com)
2. Ejecuta `supabase/schema.sql`
3. Copia las credenciales a `.env.local`

## Stack

Next.js, Dexie, Supabase, Tailwind, Recharts
